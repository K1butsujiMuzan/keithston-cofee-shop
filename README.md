# Keithston Coffee Shop

Vanilla HTML5, CSS3, JavaScript online coffee shop website. Product data stored in JSON file, cart saved in localStorage. BEM methodology for naming classes.

## Features

- Product catalog with categories
- Shopping cart with modal preview
- Cart page
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript
- JSON
- BEM

## Clone Repository

```bash
git clone https://github.com/K1butsujiMuzan/keithston-cofee-shop.git
cd keithston-cofee-shop

```
